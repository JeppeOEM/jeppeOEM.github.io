Telegram Trading Bot

Description

The goal of this project was to interact with the Binance Blockchain using the Web3py framework to be able buy and sell newly created cryptocurrencies in the cloud.

The method is to collect data about newly created currencies, which is posted on Telegram channels that only contain that.
That data is then processed and with automation posted on honeypot.is to check if itâ€™s a scam token, the result is then webscrapped, and the bot can buy the token and follows its price until it sells it again.
The whole process is Illustrated here in a activity diagram.

Challenges:

  • To ensure that the bot is selling all the tokens I bhought I had to look inside large chuncks of metadata convert it to JSON and extract the information from the nested json objects and then convert it from hexadecimals to decimals.
  • To be able to get the token address and post it on honeypot.is to check if it is a scam,
  • To be able to run the bot on Heroku while using browserless chrome I found out that adding buildpacks with links to github repositories the drivers would be installed in the cloud.

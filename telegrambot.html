Description:

A minimal viable product of a cryptocurrency trading bot which uses Web3py to send transactions and receive price data directly from the "Binance Smart Chain".
This is first step in a larger an ongoing project

The logic behind is to listen to a Telegram Channel that only post information on newly created Cryptocurrencies, and then buy these coins if certain criteria are met forexample size of liquidity pool.
One feature is a scam check that uses selenium to post the token address on a website and then web scrape the result.
If that test is passed the bot does a test buy and sell, if that is a succes. The token will be purchased and the transaction data will be stored in a sql database. The bot will then every few seconds get new price information from an endpoint until a predetermined sell price is achived. The bot has been made ready for deployment on Heroku.

Tech Stack:

Python // Pandas // Selenium // Telegram API // Web3py // SQLite3

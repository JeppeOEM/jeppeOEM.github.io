Description:

A Procedural drawing of a stiching pattern. After watching this numberphile video about i decided to create this pattern using p5.js.
The concept is to create two arrays of randomly generated populated arrays, one for each axis.
Then draw a sequence of lines and spaces either with an offset or not depending on wether the value from the array is even or uneven.
Thanks to Stackoverflow got the code got upgraded with a flood fill algorithm to add colors based on the size of the area of connected grid tiles.

React Webshop

Description:

My latest school project on delving into the world of react. The focus here is only on React not the styling.

It is a simple webshop which gets some products from a API and connects with a Supabase database to make POST requests.
I found this project extremely rewarding, the workflow with states and components makes frontend development more effecient and powerfull.
And i am amazed by how few lines of code it took to create a basic webshop template.
I like examples like below where the state is handleded like this: A whole html form including the items is passed down and updated with selected items from the cart which is really alot more effecient.

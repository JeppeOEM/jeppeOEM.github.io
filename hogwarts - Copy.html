Description:

A journey into Web scraping, Blockchain, Asynchronous programming and problem solving. A cryptocurrency trading bot which utilizes Telegram API to collects updates on new cryptocurrencies from telegram channels. One main feature is a scam check that uses selenium to post the token address on a website and then web scrape the result. If that test is passed the bot does a test buy and sell with Web3, if that is a succes.The tokens will be purchased and the transaction will be stored in sql database. The bot will then listen to a API for the price of the tokens in the database until a predetermined sell price is achived. The bot has been made ready for deployment on Heroku.

Python Pandas Selenium Telegram API Web3py

Tech Stack

  • Python
  • Pandas
  • Selenium
  • Telegram API
  • Web3.py

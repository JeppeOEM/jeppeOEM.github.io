Hogwarts Student List

Description

A study on cleaning and manipulating JSON data with many edge cases, the data is taken from the Harry Potter books. A requirement was to make a "HackTheSystem" function to mess up the JSON data after it was sorted.

The students is displayed using a html template and a list of javascript objects containing the cleaned up student data.
The sorting and filtering is controled by values contained in a html data attribute which tell what variable to filter, or to sort accending or decscending.

Features

  • Expellings students from the school and the list

  • Adding two prefect per house

  • Adding students as inquisitors but only "pure blood" inquisitors in Slytherin

  • A "HackTheSystem" function which randomly changes all the pure blood to a random status, makes it impossible to make a student an inquisitor without it changes back in a couple of seconds and injects a new hacker students that cant be expelled

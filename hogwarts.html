Hogwarts Student List

Description

A school project on cleaning and manipulating JSON data with many edge cases, the focus here is on the javascript not the design.
The data is taken from the Harry Potter books.

The students is displayed using a html template and a list of objects. Containing the cleaned up student data.
The sorting and filtering is controled by values contained in a html data attribute, which tell what to filter and what to sort, including the direction.

Features

  • Expellings students from the school

  • Adding two prefect per house

  • Case sensitive Search bar

  • Adding students as inquisitors but only "pure blood" inquisitors in Slytherin

  • A "HackTheSystem" function which randomly changes all the pure blood to a random blood status, makes it impossible to make a student an inquisitor, without it changes back in a couple of seconds, and injects a new students that cant be expelled, search for 1337 to activate.

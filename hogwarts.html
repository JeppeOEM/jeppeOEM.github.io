Hogwarts Student List

Link to webpage

A study on cleaning and manipulating JSON data with many edge cases on a imaginary wizard school.

The students is displayed using a html template and a list of javascript objects containing the cleaned up student data.
The sorting and filtering is controled by values contained in data attribute which tell what variable to filter or to sort accending or decscending forexample.

A number of diffrent manipulations are possible with the the list which include:

  • expelling students,
  • adding students as prefects but only two per house.
  • adding students as inquisitors but only "pure blood" inquisitors in Slytherin.
  • A "HackTheSystem" function which randomly changes all the pure blood to a random status, makes it impossible to make a student an inquisitor without it changes back in a couple of seconds and injects a new hacker students that cant be expelled

Tech Stack:

Javascript // HTML // CSS
